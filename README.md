# To-do List App

A simple vanilla JavaScript to-do list with localStorage persistence.

## Features

- Add tasks with the input box or by pressing Enter
- Click a task to toggle completion (strikethrough + icon swap)
- Delete tasks with the X button
- Tasks persist across page refreshes via localStorage

## Files

| File | Purpose |
|---|---|
| `index.html` | App structure |
| `styles.css` | Styling (Bootstrap 5 + custom) |
| `script.js` | App logic |
| `assets/` | Checkbox icons |

## Usage

Open `index.html` in any browser — no build step or server needed.
